### Your Friendly AI Teacher

#### Dependencies
1. Install openai api: `pip install openai`
2. Install streamlit: `pip install streamlit`
3. Install lumaai: `pip install lumaai` 

#### Run instructions
`streamlit run ./ai_teacher.py`
